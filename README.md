This is raw data captured directly from weather station deployed at Bradford-Renduchintala Centre for Space AI in University of Bradford, UK.
The File "Archieved_WeatherStationData_24_25_26.txt" contains the data captured from Nov 2024 to end of June 2026. 
The File "LiveData.txt" represents recently captured data. This file is updated every 30 second and Git repository is updated every two hours.
All the image files containing the graphs showing differetn weather attributes are also updated every two hours on this repository.
